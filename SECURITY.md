# Security and Safe Use

This repository contains a legacy training application undergoing modernization. It is not intended for production use or for processing real customer, employee, credential, payment, or financial information.

## Safe-use requirements

- Use synthetic demonstration data only.
- Do not commit passwords, tokens, private keys, or local configuration.
- Do not use generated `.dat` files from untrusted sources.
- Treat the login and payment workflows as educational examples rather than production security controls.
- Review the Java version and dependencies before running the project in a sensitive environment.

## Current status

A formal dependency, code, data-handling, and configuration review remains part of the modernization roadmap.
